# Changelog

All notable changes to the Archon VS Code extension will be documented in this file.

## [0.1.0] - 2025-01-01

### Added

- **Skills Explorer**: Categorized tree view of all 49+ skills with installation status badges
- **Skills Search**: Filter skills across all categories by name, description, or tag
- **Skill Detail Panel**: Rich webview showing SKILL.md documentation and metadata
- **Agents Panel**: Flat list of 8 agents with role descriptions and documentation viewer
- **Agent Invocation**: Right-click to invoke agents with parameter input via terminal
- **Pipelines Panel**: Tree view of 5 pipelines with expandable steps and status icons
- **Pipeline Dashboard**: Real-time webview with step progress, context budget meter, and artifact links
- **Bundles Panel**: Tree view of 8 bundles with expandable skill contents and installation status
- **Health Report**: Doctor command results with circular score gauge and platform health cards
- **Status Bar**: Pipeline status indicator and context budget meter with color coding
- **13 Commands**: Initialize, Install Skill, Install Bundle, Uninstall, Run Pipeline, Show Dashboard, Run Doctor, Validate Workspace, Search Skills, Admin Dashboard, Update, Open Settings, Show Context Budget
- **5 Settings**: cliPath, defaultPlatform, showStatusBar, autoRefreshDashboard, contextBudgetWarning
- **CLI Integration**: JSON envelope parsing, timeout handling, CLI-not-found prompts
- **Cross-Editor Support**: Compatible with VS Code, Cursor, and Windsurf
