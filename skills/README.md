# Skills

All OMNISKILL skills live here. Each skill has its own directory with:

- `SKILL.md` — Instructions the AI agent follows
- `manifest.yaml` — Metadata, triggers, dependencies
- `resources/` — Reference materials
- `examples/` — Sample interactions
- `templates/` — Output templates
- `tests/` — Validation test cases
- `overrides/` — Platform-specific overrides

## Template

See [`_template/`](_template/) for the skill template to copy when creating new skills.
